Generation Instructions

  1. Copy the SVG code above and save it as trackdeni-icon.svg
  2. Use an online SVG to PNG converter or design tool to create the following files:
    • icon-72x72.png
    • icon-96x96.png
    • icon-128x128.png
    • icon-144x144.png
    • icon-152x152.png
    • icon-192x192.png
    • icon-384x384.png
    • icon-512x512.png
  3. Place all PNG files in the public/icons/ directory
  4. Also create shortcut icons: add-customer-96x96.png and add-debt-96x96.png
